Данный файл представляет собой инструкцию по пользованию этой программой. Программа представляет собою парсером
интернет площадки "Яндекс.Маркет" по комплектующим ПК и занесение их в базу данных (в таблицы GoogleSheets)

Основные Файлы:
GoogleSheets.py - Настройка GoogleSheets под проект и подключение его к Python.
parser.py - парсер интернет площадки "Яндекс.Маркет" по комплектующим ПК и занесение их в базу данных
credentials.json - ключ API GoogleSheets
all_sheets_data.json - база даннызз комплектующих в формает JSON
my_database.db - база даннызз комплектующих в формает SQL

"Data from all sheets saved to all_sheets_data.json
Process finished with exit code 0" то переходим к шагу №2. Если видим ошибку читаем дальше.

Решение возможных ошибок:

1. "при запуске кода не открывается окно браузера и выдает ошибку"
Для решения этой проблемы вам нужно заменить chromedriver.exe. Для этого узнаете внрсию вашего браузера Chrome 
и скачиваете нужную вам версию chromedriver(https://getwebdriver.com/chromedriver), распаковаете и заменяете файл chromedriver.exe на ваш новый.
